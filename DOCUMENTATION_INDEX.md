## 📱 Responsive Behavior Reference

Quick reference for what to expect:

- **Mobile (≤480px)**: 1-column layouts, compact spacing, mobile menu
- **Tablet (481-768px)**: 2-column grids, mobile menu available
- **Large Tablet (769-1024px)**: Mixed layouts, preparation for desktop
- **Desktop (1024px+)**: 3-column grids, full spacing, all navigation

See: CSS_RESPONSIVE_VISUAL_GUIDE.md for visual representations
